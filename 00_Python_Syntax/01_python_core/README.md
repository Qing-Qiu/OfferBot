# 01 python_core

本小章节用于补齐最基础的 Python 语法地基：

- 变量、对象、引用
- 基础类型与容器
- 函数、参数、返回值
- 控制流
- 类型标注、异常、入口函数

## 学习目标

读完本节后，至少要能做到：

```text
看懂变量绑定和对象引用
看懂 list / dict / tuple / set 的常见用法
看懂函数参数、返回值和类型标注
看懂 if / for / while / try 这些控制流
能判断一段 AI demo 代码从哪里开始执行
```

## 后续展开顺序

1. 变量、对象、引用：为什么 `a = b` 不是复制全部数据。
2. 容器：list / tuple / dict / set 的选择和遍历。
3. 函数：默认参数、关键字参数、返回值。
4. 类型标注：读懂 `list[int]`、`Optional[str]`、`tuple[int, int]`。
5. 异常和入口：`try/except`、`raise`、`if __name__ == "__main__"`。
