# 03 python_code_reading

本小章节用于提升“看懂工程代码”的能力：

- 推导式
- `enumerate` / `zip` / `range`
- `lambda`
- `sorted(key=...)`
- `with`
- `_method` / `weight_` 等命名习惯

## 学习目标

本节不追求写出炫技 Python，而是训练“读代码速度”：

```text
看到推导式能还原成 for 循环
看到 enumerate / zip 能判断每轮变量含义
看到 lambda / key 能知道排序依据
看到 with 能知道资源何时释放
看到下划线命名能判断工程约定
```

## 后续展开顺序

1. 推导式与生成器表达式。
2. `enumerate` / `zip` / `range`。
3. `lambda` 与 `sorted(key=...)`。
4. `with` 和上下文管理器。
5. 常见工程命名：`_private`、`weight_`、`__call__`。
