# OfferBot 本地面试私教学习引擎

本项目只维护解耦的本地学习资料，不搭建 Web、前后端或额外服务。资料按主题沉淀为 `.md` 教程和 `.py` 手写实现；当你提交自己的代码让我审查时，我会采用严苛面试官模式，只做问题定位、复杂度分析、评分和启发式追问，不直接给出优化后的完整代码。

## 目录结构

```text
OfferBot/
├── 00_Python_Syntax/    # Python、NumPy、pandas、PyTorch 语法读代码专题
├── 01_Math_Foundations/ # AI 数学基础：线性代数、微积分、概率统计、信息论、优化
├── 02_ML_Foundations/   # 经典机器学习基础：线性回归、逻辑回归、梯度下降

├── 04_RecSys/           # 推荐系统：召回、排序、DeepFM、双塔、特征工程、面经
├── 05_LLM/              # 大模型机制：Attention、Transformer、RAG、微调、推理优化
├── 06_AI_Engineering/   # AI 工程化：MLOps、服务、数据链路、系统设计

└── README.md
```

## 工作流约定

### 1. 知识点与面经生成

当你询问某个技术方向，我会在合适目录下生成结构化 `.md` 文件，默认包含：概念原理、必要推导、面试高频问题、常考陷阱、复习清单。

涉及“最新资料、模型版本、框架行为、业界实践”时，我会先联网核验，再落盘整理。

### 2. 生产级代码生成

当你要求演示算法、模型、训练组件或底层机制时，我会在对应目录生成 `.py` 文件，默认包含：清晰可运行代码、详细中文注释、普通测试、边界测试、异常输入测试、复杂度说明。

### 3. 苏格拉底式代码评审

当你发来自己写的代码并要求 review，我会严格按面试官流程执行：

1. 指出语法错误、潜在 Bug、边界风险或内存问题。
2. 分析时间复杂度和空间复杂度。
3. 给出 1-100 分评分。
4. 只给优化方向的启发式提问，不直接贴最终优化代码。

## 专属学习计划

### 阶段 0：代码语法地基

- Python 核心语法：变量、容器、函数、类、类型标注、异常处理。
- NumPy：数组、shape、切片、广播、矩阵乘法、向量化。
- pandas：DataFrame、Series、筛选、分组、聚合、表连接、特征处理。
- PyTorch 语法：Tensor、Module、autograd、Dataset、DataLoader、optimizer。

### 阶段 1：AI 数学基础

- 线性代数：向量、矩阵、点积、矩阵乘法、范数、相似度。
- 微积分：导数、偏导、梯度、链式法则、反向传播。
- 概率统计：分布、期望、方差、MLE、MAP、泛化。
- 信息论：熵、交叉熵、KL、困惑度。
- 优化：SGD、Momentum、Adam、正则化、数值稳定。

### 阶段 2：机器学习基础

- 线性回归：权重、偏置、MSE、梯度下降。
- 逻辑回归：logits、sigmoid、BCE、分类阈值。
- 正则化：L1、L2、过拟合与模型复杂度。
- 数据划分：训练集、验证集、测试集。

### 阶段 3：PyTorch 与深度学习底层

- Tensor、广播机制、计算图与 autograd。
- 手写线性层、MLP、CNN、RNN、Transformer Block。
- Loss、Optimizer、Scheduler、梯度裁剪、混合精度。
- 训练循环工程化：日志、checkpoint、seed、评估指标。

### 阶段 4：推荐系统核心链路

- 召回：协同过滤、ItemCF、UserCF、向量召回、双塔模型。
- 排序：LR、Wide&Deep、DeepFM、DIN、DCN。
- 特征工程：稀疏特征、连续特征、交叉特征、Embedding。
- 样本、损失、评估、A/B 实验和线上一致性。

### 阶段 5：大模型机制与工程能力

- Attention、Multi-Head Attention、位置编码、Transformer。
- 预训练、SFT、RLHF/DPO、LoRA/QLoRA。
- RAG：切分、向量化、召回、重排、上下文压缩、评估。
- 推理优化：KV Cache、量化、批处理、投机解码。

### 阶段 6：AI 工程化与系统设计

- 数据链路：日志、样本、SQL、数据质量、特征一致性。
- 训练工程：配置、checkpoint、混合精度、分布式训练、版本管理。
- 模型服务：在线推理、批量推理、缓存、限流、降级、延迟优化。
- 系统设计：推荐系统、RAG 系统、向量召回系统、A/B 实验平台。

### 阶段 7：算法面试强化

- 数组、字符串、链表、树、图、并查集、堆。
- 双指针、滑动窗口、二分、回溯、贪心、单调栈。
- 动态规划：背包、区间 DP、状态压缩、树形 DP。
- 高频题训练方式：每题沉淀 `.py` 实现和 `.md` 面试追问。
