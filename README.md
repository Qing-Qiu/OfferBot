# OfferBot 本地 AI 面试私教

OfferBot 是一套面向算法、机器学习、PyTorch、推荐系统和大模型面试的本地学习资料库。内容以独立 Markdown 讲义和 Python 手写实现沉淀，MkDocs 只负责静态浏览和搜索，不引入业务 Web、前后端或额外服务。

打开文档站后，直接从左侧导航选择要学习的主题即可。每篇讲义都尽量围绕三件事展开：先讲清核心直觉，再给出必要公式或代码阅读线索，最后补上面试高频问法、常见陷阱和自测题。

## 当前覆盖

```text
已较完整：数学基础
已可开学：机器学习基础、PyTorch、推荐系统、大模型、AI 工程化

```

建议学习顺序：

1. Python 语法：先补齐读代码所需的 Python、NumPy、pandas、PyTorch 基础表达。
2. 数学基础：重点掌握 shape、梯度、概率统计、信息论、优化和矩阵求导。
3. 机器学习基础：理解线性模型、损失函数、评估指标和泛化问题。
4. PyTorch：把 Tensor、autograd、Module、训练循环和优化器串成完整闭环。
5. 推荐系统与大模型：分别进入召回排序链路、Transformer、Attention 和 RAG。
6. AI 工程化与算法：补系统设计、模型服务、工程链路和高频算法题。

## 使用约定

当你询问某个技术方向时，我会在对应主题下补充结构化讲义，默认包含原理、必要推导、面试追问、常见陷阱和自测题。涉及最新模型、框架版本或业界实践时，我会先联网核验，再整理成本地资料。

当你要求演示算法、模型或训练组件时，我会生成可直接运行的 Python 脚本，包含详细中文注释、普通测试、边界测试、异常输入测试和复杂度说明。

当你提交自己的代码让我 review 时，我会进入严苛面试官模式：指出问题，分析时间与空间复杂度，给出 1-100 分评分，并只用启发式问题引导你优化，不直接贴最终答案。
