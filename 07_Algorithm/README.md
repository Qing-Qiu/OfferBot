# 07 Algorithm：算法面试强化

本章放在最后，但不代表最后才开始练。算法更像体能训练，建议和 AI 主线并行推进。

## 学习目标

```text
能稳定写出正确代码
能解释时间复杂度和空间复杂度
能识别题型范式
能在面试中讲清思路、边界和优化
```

## 每日训练建议

```text
20 分钟：复盘一个旧题模板
40 分钟：限时做一道中等题
20 分钟：补边界 case 和复杂度
10 分钟：写面试口述版思路
```

## 高频路线

1. 数组、字符串、哈希表。
2. 双指针、滑动窗口、前缀和。
3. 二分查找与答案二分。
4. 栈、队列、单调栈、堆。
5. 链表、二叉树、图、并查集。
6. 回溯、贪心、动态规划。
7. Hard 题范式：状态设计、剪枝、单调结构、图建模。

## 代码评审规则

你提交自己的算法代码时，我会按严苛面试官模式：

```text
1. 指出语法错误、潜在 bug、边界风险
2. 分析时间复杂度和空间复杂度
3. 给 1-100 分评分
4. 只给启发式优化提问，不直接贴最终正确代码
```
