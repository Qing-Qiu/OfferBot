# 02 ML Foundations：机器学习基础

这个目录用于存放经典机器学习基础内容，例如线性回归、逻辑回归、损失函数、梯度下降、正则化、训练/验证/测试划分等。

## 为什么单独建这个目录

线性回归、逻辑回归这些内容严格来说属于经典机器学习，不是深度学习专属。

但它们又是深度学习的地基，因为深度学习也沿用同一套训练语言：

```text
参数 -> 前向计算 -> 损失函数 -> 梯度 -> 优化器更新 -> 泛化评估
```

区别主要在于：

- 经典机器学习模型通常结构更浅、更可解释。
- 深度学习模型通常由多层神经网络组成，参数更多，表达能力更强。
- 二者都可以有权重、偏置、损失函数、梯度和优化过程。

## 学习顺序建议

1. 线性回归：理解权重、偏置、MSE、梯度下降。
2. 逻辑回归：理解 logits、sigmoid、BCE、分类阈值。
3. 正则化：理解 L1、L2、过拟合控制。
4. 训练评估：理解训练集、验证集、测试集、泛化误差。
5. 再进入 `03_PyTorch`：用 PyTorch 表达同样的训练闭环。
