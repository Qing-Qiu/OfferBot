# 06 AI Engineering：AI 工程化与系统设计

本章用于连接“会训练模型”和“能把模型稳定落地”之间的差距。

核心目标：

```text
能设计 AI 系统
能解释训练到上线的完整链路
能定位线上效果和性能问题
能回答大厂工程化与系统设计追问
```

## 覆盖范围

- MLOps：训练、评估、部署、监控、回滚。
- 数据工程：日志、样本、SQL、数据质量、特征一致性。
- 模型服务：在线推理、批量推理、延迟、吞吐、限流、降级。
- 训练工程：checkpoint、混合精度、分布式训练、显存优化。
- 向量检索：Faiss、HNSW、Milvus、索引更新。
- 系统设计：推荐系统、RAG 系统、搜索排序系统、广告系统。
- 线上实验：A/B 测试、指标、显著性、灰度发布。

## 学习方式

每个系统设计题都按同一模板沉淀：

```text
1. 业务目标
2. 数据链路
3. 模型链路
4. 服务架构
5. 指标与实验
6. 性能瓶颈
7. 风险与兜底
8. 面试追问
```
